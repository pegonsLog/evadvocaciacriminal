# Product Overview

EV Advocacia Criminal is a client management system for a criminal law practice. The application manages:

- **Client Information**: Personal details, contact information, and case data
- **Payment Control**: Installment tracking, payment processing, and financial records
- **User Management**: Role-based access with admin and regular user permissions
- **Authentication**: Firebase-based user authentication and authorization

The system is designed for a small law firm to track client payments, manage installment plans, and maintain organized client records with proper access controls.

## Key Features

- Client registration and management
- Installment payment tracking
- Payment history and status monitoring
- Role-based user access (Admin/Regular users)
- Firebase integration for data persistence and authentication
